# ChatGPT-Reasoning-Shortcut
Nobody wants to be doing extra working when you're outsourcing your thinking to a LLM. Isn't it annoying when you have to click 2 buttons to enable thinking mode for every single prompt? Well, this browser extension fixes that :3

You can install it from the [Edge Add ons store](https://microsoftedge.microsoft.com/addons/detail/thinkle/iaoneianckmpfdlojdbfhnifikabmmpm)
The default shortcut is ctrl +q, but you can set to whatever you like. The textbar will be surrounded by a blue outline with thinking is enabled. 
While the extension works fine for the most part, it cannot detect when the user hits their limit of number of reasoning prompts, and you will have to send the prompt again with reasoning off. 
Hopefully the extension doesn't break when an update rolls out xD
